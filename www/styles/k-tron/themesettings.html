Theme details

The default theme is a safe theme that you can always fall back on. It comes with the following features:

Dashboard Columns

This feature will create vertical columns on your dashboard, one for each category. BUT this feature only springs into action if you have enough horizontal space for the amount device-types that you have. So for 5 columns you need at least a 1300 pixels wide screen, for example. This feature works for both the normal and compact display option.

Settings Sidemenu

Instead of the tabs at the top of each settings page, you get a sidebar instead. It stays in place. This includes the 'save' button, so you always have easy access to that.

Merge Items

Follow this pattern when naming your utilities to have them be merged: groupname - value. So for example the items "Plant 1 - moisture" and "Plant 1 - fertility" will be merged. This feature will only merge simple items with one data output.

Dashboard Data Visualizations

This will allow javascript overlays a datavisualisation for certain items. Percentages get visualised. Some items, like temperature, even get complete charts for the past 24 hours.

Theme settings

Vertical Columns on dashboard

This will show items on the dashboard in vertical columns, instead of horizontal stacking.
Vertical menu for settings

This will turn the tabs at the top of this page into a vertical menu on the left which stays in place, allowing for quick access.
Show Last Update time

This will show the time that an item was last updated.
Sunrise times at the top

Show the date and the sunrise and sunset times in the footer instead of the top of the page.
Merge utilities

This merges utility items with a similar name into one dashboard item.
Show Data Visualisations

Some highlighted items on the dashboard will have a simple data visualisation.
